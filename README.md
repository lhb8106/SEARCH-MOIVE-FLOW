## 📌 메인 기능
#### 영화 검색
- 네이버 검색 API 사용
- Paging3를 통한 검색 결과 페이징 처리
- 웹뷰를 통한 영화 검색 상세 페이지 이동

#### 최근 검색어 저장
- room을 통한 검색어 저장
- 검색어 전체 삭제
